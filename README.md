## 技能配置
赛前技能配置文件：`src/marbleSkills.json`

- 每个弹珠可配置不同的赛前被动技能
- 配置文件包含中文备注字段，方便直接阅读和调整
- 当前技能以弹珠玩法相关规则增强为主，例如“可穿墙1次”“可强化反弹2次”
- 对局开始时会按当前弹珠读取一次配置并生成本局技能快照
- 对局中不会实时跟随弹珠切换而变化
